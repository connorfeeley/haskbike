* Useful Commands
** Run Tests (Showing all Output)

#+begin_src shell :noeval
cabal test --test-show-details=direct
#+end_src

** Update Dependencies

#+begin_src shell :noeval
cabal freeze --enable-tests --upgrade-dependencies --allow-newer=aeson
#+end_src

** Query Database

#+begin_src shell :noeval
psql -d haskbike -c "SELECT * FROM station_information"
#+end_src

** Create Database

#+begin_src shell :noeval
createdb haskbike
#+end_src
